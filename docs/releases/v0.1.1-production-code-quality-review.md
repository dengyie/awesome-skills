# v0.1.1: Production Code Quality Review Workflows

## Summary

This release focuses on productizing the `production-code-quality-review` skill experience.

It adds:

- a golden path for first-time users
- a compact review brief mode
- install and update helper scripts
- release and troubleshooting docs
- fixture-style coverage for representative repo shapes

## What Changed

### Onboarding

Added a shortest-path guide for getting from install to a useful review brief in the fewest steps.

### Review Brief Output

Added `compact` output mode to `review-entrypoint.py` for fast scanning and lightweight handoff.

### Maintenance Workflow

Added helper scripts for local install and local update of the skill package.

### Reliability

Added fixture-style tests that cover:

- TypeScript API style repos
- database migration changes
- Docker/runtime changes

### Documentation

Added:

- `CHANGELOG.md`
- release checklist
- troubleshooting guide

## Validation

Validated with:

```bash
python3 -m unittest discover production-code-quality-review/tests -v
python3 production-code-quality-review/scripts/review-entrypoint.py --repo . --format compact
python3 production-code-quality-review/scripts/install-local-skill.sh
python3 production-code-quality-review/scripts/update-local-skill.sh
```

## Notes

This release builds on `v0.1.0` and keeps the core review philosophy unchanged.
