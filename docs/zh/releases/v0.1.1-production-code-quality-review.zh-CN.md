# v0.1.1：Production Code Quality Review Workflows

## 摘要

这个版本聚焦于把 `production-code-quality-review` 的使用体验做得更像一个成品。

主要新增：

- 面向首次使用者的黄金路径
- 紧凑版 review brief 输出模式
- 本地安装与更新辅助脚本
- release 与 troubleshooting 文档
- 覆盖代表性仓库形态的 fixture 风格测试

## 具体变化

### Onboarding

新增一条最短路径，帮助用户从安装直接走到第一份有价值的 review brief。

### Review Brief 输出

`review-entrypoint.py` 新增 `compact` 输出模式，适合快速扫描和轻量 handoff。

### 维护工作流

新增本地安装与本地更新辅助脚本，方便日常维护 skill 包。

### 可靠性

新增 fixture 风格测试，覆盖：

- TypeScript API 风格仓库
- 数据库迁移变更
- Docker / runtime 变更

### 文档

新增：

- `CHANGELOG.md`
- release checklist
- troubleshooting guide

## 验证

```bash
python3 -m unittest discover production-code-quality-review/tests -v
python3 production-code-quality-review/scripts/review-entrypoint.py --repo . --format compact
python3 production-code-quality-review/scripts/install-local-skill.sh
python3 production-code-quality-review/scripts/update-local-skill.sh
```

## 备注

这个版本建立在 `v0.1.0` 之上，核心 review 哲学没有变化。
